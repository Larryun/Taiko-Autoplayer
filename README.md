# Taiko Autoplayer

An simple auto play script for Taiko the web game.
### 99% Accuracy!

Writen in python3.6

## Get Started
### Install Required library
```text
numpy
mss
opencv-python
pynput
```
Run the script in 1920x1080 resolution on Windows.
Full screen your browser(Chrome is preferred).
